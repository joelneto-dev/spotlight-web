# Blog do Spotlight — TODO v2.0

Diretório preparado para a criação do blog acadêmico oficial do TCC.

## Planejamento de Implementação

- **Tecnologia:** Uso do Next.js MDX (`@next/mdx`) para carregar artigos escritos diretamente em markdown.
- **Conteúdo:**
  - Diário de bordo do desenvolvimento do TCC.
  - Artigos explicando busca semântica, processamento de embeddings e NLP.
  - Tutoriais e discussões de arquitetura.
- **Estilo:** Estilo de leitura limpo e minimalista usando `@tailwindcss/typography`.
